Module 11: HTML Formatting

1. Introduction to HTML Formatting:

In HTML, text formatting plays a significant role in enhancing the visual presentation and readability of web pages. By applying various formatting options, you can emphasize important content, create visual hierarchy, and improve the overall user experience. This module introduces several HTML tags and attributes that allow you to format text effectively.

2. Bold Text:

3. Italic Text:

The and tags are used to italicize text. Similar to bold tags, the tag carries more semantic meaning and should be used for text with emphasized importance. The tag is used for stylistic purposes only, indicating text that is set in an alternative font style.

4. Underline Text:

The tag is used to underline text. However, underlining text is generally discouraged for body content as it is conventionally associated with hyperlinks. Instead, use CSS to apply underline styles to specific elements, such as hyperlinks.

5. Strikethrough Text:

The tag is used to apply a strikethrough effect to text. It visually indicates that the text is no longer valid or relevant. Strikethrough is commonly used for indicating deleted or outdated information.

6. Superscript and Subscript Text:

The and tags are used to create superscript and subscript text, respectively. Superscript text appears above the baseline, while subscript text appears below the baseline. These tags are often used for mathematical expressions, chemical formulas, and footnotes.

8. Semantic Text Styling:

When applying text formatting, it's essential to use tags that convey semantic meaning. Use , , and other semantic tags to add emphasis, importance, or special meaning to text. This practice not only enhances the visual appearance but also improves accessibility for screen readers and other assistive technologies.

9. Text Styling and Accessibility:

When styling text, consider accessibility to ensure that all users can access and comprehend the content. Avoid relying solely on color for conveying information. Ensure there is sufficient contrast between text and background colors for readability. Use CSS to adjust font sizes, line spacing, and letter spacing to improve accessibility.

  • Implement a design that uses appropriate text formatting to create a clear visual hierarchy.
