Module 15: HTML CSS

1. Introduction to Cascading Style Sheets (CSS):

Cascading Style Sheets, commonly known as CSS, is a style sheet language used to control the presentation and layout of HTML documents. CSS enables you to define styles for HTML elements, such as fonts, colors, spacing, and positioning, without modifying the HTML structure. By separating the presentation from the content, CSS promotes clean, organized, and maintainable code.

2. Integrating CSS with HTML:

CSS can be integrated with HTML documents in several ways. The most common methods are:

  • Inline CSS: Adding CSS styles directly within the HTML elements using the "style" attribute.
  • Internal CSS: Defining CSS styles within the "style" element in the HTML section.
  • External CSS: Linking an external CSS file to the HTML document using the "link" element.
        <!-- Inline CSS -->
        <p style="color: blue; font-size: 18px;">This is a paragraph with inline CSS styles.</p>

        <!-- Internal CSS -->
        <head>
            <style>
                p {
                    color: blue;
                    font-size: 18px;
                }
            </style>
        </head>

        <!-- External CSS -->
        <head>
            <link rel="stylesheet" href="styles.css">
        </head>

3. CSS Selectors:

CSS uses selectors to target HTML elements and apply styles to them. Selectors can be based on element names, class names, IDs, attributes, and more. Understanding CSS selectors is fundamental to styling specific elements or groups of elements. 

        <!-- Element Selector -->
        <p>This is a paragraph targeted using the element selector.</p>

        <!-- Class Selector -->
        <p class="highlight">This paragraph has a class selector applied.</p>

        <!-- ID Selector -->
        <p id="intro">This paragraph has an ID selector applied.</p>

4. CSS Box Model:

The CSS box model is a fundamental concept that defines how elements are structured and sized in CSS. Each element is treated as a rectangular box with four components: content, padding, border, and margin. Understanding the box model is crucial for controlling element size, spacing, and layout on a web page. 

        <style>
            .box {
                width: 200px;
                height: 100px;
                padding: 10px;
                border: 2px solid black;
                margin: 20px;
            }
        </style>

        <div class="box">
            This is a box element with padding, border, and margin.
        </div>

5. Advanced Styling with CSS:

CSS provides various advanced styling features, such as:

  • Font and Text Styling: Controlling font family, size, weight, style, and text alignment.
  • Backgrounds and Gradients: Setting background colors, images, and creating gradient effects.
  • Positioning and Layout: Positioning elements using absolute, relative, and fixed positioning, and creating responsive layouts.
  • Transitions and Animations: Adding smooth transitions and animations to elements for enhanced user experience.
  • Media Queries: Implementing media queries to apply different styles based on the user's device screen size.

6. Practical Application:

Create a simple web page and apply different CSS styles to customize the appearance. Experiment with various selectors and properties to gain hands-on experience with CSS.

7. Challenges (For experienced students):

  • Create a responsive website layout using media queries to adapt to different screen sizes.
  • Implement CSS animations to add engaging visual effects to elements on a web page.
