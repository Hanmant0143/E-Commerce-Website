HTML Introduction

HTML is the acronym for Hyper Text Markup Language. It serves as the standard markup language for creating web pages and defining their structure. HTML is the backbone of web development and enables web browsers to interpret and display content correctly.

An HTML document is built using a series of elements, each labeled with specific tags that instruct the browser on how to display the content. These elements include headings, paragraphs, links, images, tables, and more, which collectively compose the visible content of a web page.

A Simple HTML Document


    <!DOCTYPE html>
    <html>
    <head>
      <title>Page Title</title>
    </head>
    <body>

      <h1>My First Heading</h1>
      <p>My first paragraph.</p>

    </body>
    </html>

The above example is a basic HTML document. Here's an explanation of its components:

  • <!DOCTYPE html>: This declaration indicates that the document is written in HTML5.
  • <html>: The root element of an HTML page, encompassing the entire document.
  • <head>: This section contains meta information about the HTML page, including the page's title displayed in the browser's title bar or tab.
  • <title>: Defines the title of the HTML page.
  • <body>: Contains all the visible contents of the page, such as headings, paragraphs, images, links, etc.
  • <h1>: Represents a top-level heading, typically used for the main heading of a page.
  • <p>: Represents a paragraph element to structure and present text content.

HTML History

HTML has undergone various versions since the inception of the World Wide Web, with each version bringing improvements and new features. The evolution of HTML is as follows:

Year Version
1989 Tim Berners-Lee invented the World Wide Web (WWW).
1991 Tim Berners-Lee created HTML.
1993 Dave Raggett drafted HTML+.
1995 HTML Working Group defined HTML 2.0.
1997 W3C Recommendation - HTML 3.2.
1999 W3C Recommendation - HTML 4.01.
2000 W3C Recommendation - XHTML 1.0.
2008 WHATWG HTML5 First Public Draft.
2012 WHATWG HTML5 Living Standard.
2014 W3C Recommendation - HTML5.
2016 W3C Candidate Recommendation - HTML 5.1.
2017 W3C Recommendation - HTML5.1 2nd Edition.
2017 W3C Recommendation - HTML5.2.

This tutorial follows the latest HTML5 standard.
