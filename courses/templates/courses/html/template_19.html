Module 20: HTML Tables

1. Introduction to HTML Tables:

Tables in HTML are used to organize and present tabular data in rows and columns. They are valuable for displaying structured information in a clear and visually appealing format. HTML tables consist of the <table> element, which contains one or more rows represented by the <tr> element, and each row contains one or more cells represented by the <td> (data cell) or <th> (header cell) element.

2. Creating Basic Tables:

To create a basic table, use the following structure:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
    <th>Header 3</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    <td>Data 3</td>
  </tr>
</table>

Replace "Header 1," "Header 2," and "Header 3" with the column headers, and "Data 1," "Data 2," and "Data 3" with the corresponding data cells.

3. Merging Cells:

You can merge cells in a table to create more complex layouts. To merge cells horizontally, use the "colspan" attribute on a or element, indicating how many columns it should span. To merge cells vertically, use the "rowspan" attribute, indicating how many rows it should span.

<table>
  <tr>
    <th>Header 1</th>
    <th colspan="2">Header 2 and 3</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    <td>Data 3</td>
  </tr>
</table>

4. Adding Captions:

You can include a caption for your table using the element. The caption provides a brief description of the table's content and its purpose.

<table>
  <caption>Table Caption</caption>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

5. Applying Styles to Tables:

You can use CSS to style your tables and make them visually appealing. Apply styles to elements such as <table>, <tr>, <th>, and <td> to control colors, borders, spacing, and more.

<style>
  table {
    border-collapse: collapse;
    width: 100%;
  }
  th, td {
    border: 1px solid black;
    padding: 8px;
    text-align: center;
  }
  th {
    background-color: #f2f2f2;
  }
  </style>

6. Best Practices for Tables:

  • Use tables for displaying tabular data and avoid using them for page layout.
  • Keep tables simple and easy to read, especially when presenting large datasets.
  • Add concise and informative captions to provide context for the table's content.
  • Make sure the table is accessible by providing alternative text for table elements and headers.
  • Test the table's responsiveness on different devices and screen sizes to ensure it displays well.

7. Practical Exercises:

  • Create a table to display a list of products with their names, prices, and descriptions.
  • Experiment with merging cells to create more complex table layouts.
  • Add CSS styles to the table to customize its appearance.

HTML tables are powerful tools for organizing and presenting data in a structured format. By understanding how to create tables, merge cells, add captions, and apply styles, you can create visually appealing and informative tables that enhance the overall user experience on your website.
