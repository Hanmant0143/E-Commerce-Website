Module 26: HTML JavaScript

1. Introduction to JavaScript in HTML:

JavaScript is a versatile and powerful programming language that is commonly used to add interactivity and dynamic behavior to web pages. In HTML, you can include JavaScript code directly within the document using script tags (<script>). JavaScript allows you to respond to user actions, manipulate the DOM (Document Object Model), validate forms, and perform various other tasks that enhance the user experience.

2. Adding JavaScript to HTML:

To include JavaScript code in an HTML document, place the script tags within the <head> or <body> section. The src attribute is used to link an external JavaScript file if your code is in a separate file.

    <head>
      <script src="script.js"></script>
    </head>
  
    <body>
      <script>
        // Your JavaScript code goes here
      </script>
    </body>

3. Basic JavaScript Examples:

Here are some basic JavaScript examples that demonstrate its interactivity:

  • Displaying an Alert: Show an alert box with a message when the page loads.
    • <script>
      window.onload = function() {
        alert("Welcome to our website!");
      };
    </script>
  • Handling Button Click: Change the text of a button when it is clicked.
    • <button id="clickButton" onclick="changeText()">Click Me</button>
    <script>
      function changeText() {
        document.getElementById("clickButton").innerHTML = "Clicked!";
      }
    </script>
  • Input Validation: Validate a form field before submitting the form.
    • <form onsubmit="return validateForm()">
      <input type="text" id="name" required>
      <input type="submit" value="Submit">
    </form>
    <script>
      function validateForm() {
        var nameInput = document.getElementById("name").value;
        if (nameInput === "") {
          alert("Please enter your name.");
          return false;
        }
        return true;
      }
    </script>

4. Best Practices:

  • Always include your JavaScript code within script tags to avoid compatibility issues.
  • Place script tags at the end of the body section to ensure that the HTML content is loaded before running the JavaScript, which can improve page performance.
  • Use descriptive function and variable names to make your code more readable and maintainable.
  • Regularly test your JavaScript code across different browsers to ensure cross-browser compatibility.

5. Practical Exercises:

  • Create a simple quiz with JavaScript that displays questions and calculates the score.
  • Add form validation to a contact form to ensure all required fields are filled before submission.
  • Implement a responsive image gallery that changes images when the user clicks on thumbnails.

JavaScript is a fundamental part of web development, enabling dynamic and interactive web pages. By understanding the basics of JavaScript, you can create more engaging and user-friendly experiences for your website visitors.
