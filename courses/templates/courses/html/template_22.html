Module 23: HTML Classes

1. Introduction to HTML Classes:

In HTML, classes are used to apply specific styles to one or more HTML elements. Classes provide a way to group elements and target them with CSS for consistent styling across the webpage. By assigning classes to elements, you can easily apply custom styles to individual elements or groups of elements without affecting other parts of the page.

2. Creating a Class:

To create a class, you use the "class" attribute on an HTML element and give it a name. The name should be unique and meaningful to describe the purpose of the class. For example:

<p class="highlighted-text">This is a paragraph with a custom class.</p>

3. Applying Styles with CSS:

Once you have created a class, you can define its styles using CSS. In a separate CSS file or within a <style> block, use the class selector (preceded by a period) to target elements with that class. For example:

<style>
.highlighted-text {
  background-color: yellow;
  font-weight: bold;
}
</style>

4. Applying Multiple Classes:

An element can have multiple classes, allowing you to combine different styles. Separate class names with a space when applying multiple classes to an element. For example:

<p class="highlighted-text large-font">This paragraph has multiple classes.</p>

5. Class vs. ID:

While classes are used to apply styles to multiple elements, IDs are used to uniquely identify individual elements. Unlike IDs, you can apply the same class to multiple elements throughout the page.

6. Reusability and Consistency:

Using classes promotes code reusability and ensures consistency in the appearance of similar elements. By defining styles once in a class, you can easily apply those styles to multiple elements as needed.

7. Best Practices:

  • Use classes to group elements that share similar styles or serve the same purpose.
  • Give classes meaningful names to make the code more readable and maintainable.
  • Avoid using inline styles and use classes for better separation of concerns.
  • Combine classes with other selectors (e.g., tag names or IDs) to create specific and targeted styles.

8. Practical Exercises:

  • Create a class for buttons and apply styles for consistent button appearance throughout the website.
  • Use classes to style different types of alerts (e.g., success, warning, error) with distinct colors and icons.
  • Experiment with combining multiple classes to apply compound styles to specific elements.

Using classes effectively in HTML and CSS allows you to achieve consistent styling and better organization of your webpage's elements. By following best practices and combining classes with other selectors, you can create visually appealing and user-friendly web pages.
