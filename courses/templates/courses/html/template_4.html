Module 5: HTML Basic

1. HTML Documents:

  • All HTML documents must begin with a document type declaration: <!DOCTYPE html>.
  • The HTML document structure starts with the <html> tag and ends with </html>.
  • The visible content of the HTML document is placed between the opening and closing <body> tags.

Example:


<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

2. The <!DOCTYPE> Declaration:

  • The <!DOCTYPE> declaration represents the document type and ensures proper rendering by browsers.
  • It should only appear once at the beginning of the HTML document.
  • The <!DOCTYPE> declaration is not case-sensitive.
  • For HTML5 documents, the declaration is simply: <!DOCTYPE html>.

3. HTML Headings:

  • HTML headings are defined using <h1> to <h6> tags.
  • <h1> defines the most important heading, while <h6> defines the least important.

Example:


<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

4. HTML Paragraphs:

Paragraphs are defined using the <p> tag.

Example:


<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

5. HTML Links:

Links are created using the <a> tag.

Example:


<a href="https://www.example.com">This is a link</a>

6. HTML Images:

Images are inserted using the <img> tag, which requires the src attribute to specify the image file path and the alt attribute to provide alternative text.

Example:


<img src="image.jpg" alt="An example image" width="300" height="200">

7. How to View HTML Source:

To view the HTML source code of a webpage:

  • Right-click on the page and select "View Page Source" (in Chrome) or "View Source" (in Edge or other browsers).
  • A new window will display the HTML source code of the page.

8. Inspecting HTML Elements:

To inspect elements on a webpage and view both the HTML and CSS:

  • Right-click on an element (or a blank area) and choose "Inspect" or "Inspect Element."

In this module, you will gain essential knowledge of HTML's core concepts and learn to create basic web pages. As you progress, you will explore more advanced HTML features and unlock the power of web development.
